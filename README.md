# My Shell Kit

This repository contains shell configuration files and scripts for various operating systems. Each folder corresponds to a specific platform:

- **fedora/**: Shell configuration for Fedora Linux.
- **windows/**: Shell configuration for Windows.
- **wsl/**: Shell configuration for Windows Subsystem for Linux.
- **macos/**: Shell configuration for macOS.

Please check the README files in each folder for more details on the configurations for each platform.
